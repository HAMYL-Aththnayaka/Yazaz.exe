## 🚀 Getting Started

### Prerequisites

* Node.js (LTS recommended)
* npm or yarn

### Installation

1. Clone the repository

```bash
git clone https://github.com/HAMYL-Aththanayaka/Yazaz.exe.git
```

2. Go to project folder

```bash
cd Yazaz.exe/frontend
```

3. Install dependencies

```bash
npm install
```

4. Run development server

```bash
npm run dev
```
